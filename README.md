# DEVOPS.HUB

**Master DevOps. Crack Interviews. Get Hired.**

## Overview

DEVOPS.HUB is a full-stack 3-tier web application created primarily for DevOps learning, practice, and portfolio development.

The project focuses on understanding how a real-world application is structured across multiple layers and how DevOps engineers can build, deploy, automate, monitor, and manage such applications.

This project was developed with the assistance of _AI tools_ and was used as a hands-on learning project to practice DevOps concepts including application deployment, containerization, CI/CD, infrastructure automation, cloud integration, and operational workflows.

## Architecture

This project follows a strict 3-tier architecture:

Frontend
↓
Backend API
↓
Database

### Components

* Frontend: React + TypeScript + Tailwind CSS
* Backend: Node.js + Express + TypeScript
* Database: PostgreSQL
* ORM: TypeORM

## Project Structure

frontend/
backend/
database/

Each layer is maintained separately to simulate an enterprise-style application architecture.

## Important Note

This project was created primarily as a DevOps practice and learning project.

A significant portion of the application structure, boilerplate code, documentation, and implementation was generated with the assistance of AI tools and then deployed, configured, tested, debugged, and managed as part of hands-on DevOps learning activities.

The main objective is to gain practical experience in:

* Application deployment
* Infrastructure management
* Containerization
* CI/CD implementation
* Production-style workflows

rather than showcasing advanced software engineering expertise.
